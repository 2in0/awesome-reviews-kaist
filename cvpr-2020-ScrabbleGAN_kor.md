# ScrabbleGAN: Semi-Supervised Varying Length Handwritten Text Generation\[Kor\]

##  1. Problem definition



ScrabbleGAN 논문은 CVPR 2020에 개제된 논문이다. Handwritten Text Generation을 주제로 하고있으며, GAN구조에 Recognition을 할 수 있는 모델을 제안하였다.  이 논문의 주요 Contribution으로는 손글씨 특성 상 출력의 크기가 일정하지않아 기존  Handwritten Text Generation에 쓰이는 RNN-based모델이 아닌, Fully Convolutional Neural Network를 제안했다는 점,  unlabeled data에 대해 Semi-supervised learning을 시도했다는 점, 그리고 해당 모델을 기존 데이터셋과 추가적으로 구성함으로써 데이터셋의 다양성을 확보해 기존 Handwritten Text Recognition(HTR) 모델의 성능을 올렸다는 점이다.
